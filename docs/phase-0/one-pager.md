# Magenta TC

**An opinionated transaction-coordination system for top-producing agents and small teams.**

Built by working agents, for working agents. Saves 8+ hours a week on file management. Pays for itself if you do 2 deals a month.

---

## The problem we built it for

If you do 12+ deals a year, you have one of three painful TC realities:

- **You do it yourself.** 6–8 hours per file, eating into prospecting time. Mistakes cost you referrals.
- **You hired a 1099 TC service.** $200–500/file, decent quality, but you have zero visibility into what's happening day-to-day.
- **Your brokerage handles it.** Slow, generic, you're one of 200 agents in their queue.

None of these scale. We built Magenta TC because we hit the wall ourselves and couldn't find anything that fit.

---

## What Magenta TC does

### 1. Every file gets a 48-step buyer or 47-step seller checklist, automatically
The full lifecycle from contract acceptance to recording, encoded as tasks. No more "what was I supposed to do today?" Same checklist every TC follows for every file. Same emails go out at the same milestones.

### 2. One master daily task list, filtered to what's actually due
Buyer TC sees their tasks. Seller TC sees theirs. Sortable by due date, status, or file. No more file-by-file mental swap.

### 3. The 36 emails that need to go out, automatically
Pre-acceptance, post-EMD, pre-close, post-close — every templated email lives in the system. Fires on the right day to the right party.

### 4. Per-file Drive folder with subfolders, ready to go
01-Contract, 02-Disclosures, 03-Inspection, 04-Addenda, 05-Closing, 06-Broker Compliance. Created the moment a new file is submitted. No more "where do I save this?"

### 5. Real-time TC alerts to the team
Urgent files flagged 🚨. Standard files routed to the right TC. Daily refresh of the per-file checklist so anyone opening a folder sees current status.

---

## How it works (in 30 seconds)

1. **Agent submits a new file** through a simple intake form (1 min)
2. **System creates the file** — Drive folder + subfolders, sheet row, 48-step task list, per-file checklist, and TC alerts (10 sec automated)
3. **TC works the daily task list** — checks off as work happens, fills in due dates, leaves notes
4. **Automated emails fire** to the right party at the right time
5. **Agent has visibility** into where the file stands, every step
